Política de Privacidade

Última atualização: 27 de Janeiro de 2026

O aplicativo Santo Rosário · Método Montfort foi desenvolvido como um aplicativo gratuito/comercial. Este SERVIÇO é fornecido sem custo e destina-se a ser utilizado no estado em que se encontra.

Coleta e Uso de Dados

Para uma melhor experiência, ao usar nosso Serviço, podemos exigir que você nos forneça certas informações de identificação pessoal (como Nome e E-mail para criação de conta). As informações que solicitamos serão retidas por nós e usadas conforme descrito nesta política de privacidade.

O aplicativo utiliza serviços de terceiros que podem coletar informações usadas para identificá-lo:

  • Google Firebase (Para autenticação e banco de dados seguro)

Dados de Oração

Seus dados de progresso (dias rezados, contadores) são armazenados de forma segura no banco de dados Firebase e vinculados à sua conta para permitir a sincronização entre dispositivos. Não compartilhamos esses dados com terceiros.

Segurança

Valorizamos sua confiança em nos fornecer suas Informações Pessoais, portanto, estamos nos empenhando para usar meios comercialmente aceitáveis de protegê-las.

Contato

Se você tiver dúvidas ou sugestões sobre nossa Política de Privacidade, não hesite em nos contatar pelo GitHub do projeto.
